# Projeto Spring Boot

Este é um projeto baseado em **Spring Boot**, desenvolvido com as tecnologias mais recentes para construção de aplicações web seguras e robustas.

## Tecnologias Utilizadas

- **Java**: 21  
- **Spring Boot**: 3.5.6  
- **Spring Boot DevTools**: para recarga automática durante o desenvolvimento  
- **Spring Web**: construção de APIs REST  
- **Spring Security**: autenticação e autorização  
- **Validation**: validação de dados de entrada  
- **PostgreSQL**: banco de dados relacional  
- **Swagger/OpenAPI**: documentação e testes de APIs  
- **Lombok**: redução de boilerplate com getters, setters, constructors, etc.

## Funcionalidades

- Criação de APIs RESTful para gerenciamento de recursos.  
- Validação de dados de entrada usando anotações do **Spring Validation**.  
- Configuração de segurança com **Spring Security**.  
- Integração com **PostgreSQL** para persistência de dados.  
- Documentação automática das APIs via **Swagger**.  

## Requisitos

- **Java 21**  
- **Maven** 
- **PostgreSQL** em execução
